# CodeAlpha Backend Development Internship Tasks

This repository contains the projects completed during my Backend Development Internship at CodeAlpha. Each task demonstrates the application of Node.js, Express, and MongoDB to solve real-world problems.

## 🚀 Projects Overview

### Task 1: URL Shortener
A full-stack application that takes long URLs and generates shortened versions. 
- **Tech Stack:** Node.js, Express, MongoDB, EJS
- **Features:** Custom short codes, redirection logic, and database integration.

### Task 2: Event Registration System
A professional system for managing event sign-ups.
- **Tech Stack:** Node.js, MongoDB, CSS3 (Responsive Design)
- **Features:** User registration form, database storage, and mobile-friendly UI.
