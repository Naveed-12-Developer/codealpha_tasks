# Task 2: Event Registration System

A professional system for users to register for upcoming events and store their details securely.

## 🛠️ Features
- User-friendly registration form.
- Form validation and error handling.
- Data storage and retrieval from MongoDB.
- Responsive UI designed with CSS3.

## 🚀 Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Frontend:** HTML5, CSS3, JavaScript

## ⚙️ How to Run
1. Navigate to this folder.
2. Run `npm install`.
3. Ensure your MongoDB server is running.
4. Run `npm start` or `node index.js`.
