# Task 1: URL Shortener

A backend application that takes long URLs and converts them into short, easy-to-share links.

## 🛠️ Features
- Generates unique short codes for long URLs.
- Automatic redirection to the original URL.
- Persistent storage using MongoDB Atlas.
- Dynamic frontend using EJS.

## 🚀 Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Frontend:** EJS, CSS3

## ⚙️ How to Run
1. Navigate to this folder.
2. Run `npm install` to install dependencies.
3. Add your MongoDB URI in the connection string.
4. Run `npm start` or `node app.js`.
